# indicadores_economicos_web
Aplicação web (React/Vite) para visualização de indicadores econômicos, com painéis SignalEconomics, SignalRetail e estrutura setorial (Agro, Indústria, Energia). Interface web do ecossistema Indicadores Econômicos, focada em dashboards analíticos, exploração de séries e tomada de decisão orientada por dados.
